# OCR Document Reader

This project is an OCR (Optical Character Recognition) Document Reader designed to extract text from scanned documents and images.

## Features
- Extracts text from images and PDF files
- Supports multiple languages (configure as needed)
- User-friendly interface
- Outputs text in various formats (TXT, PDF, etc.)
